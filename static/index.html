Verifiable Credentails

This web application is to demo various use cases of Microsoft Entra Verified ID. Below are the few examples among many:

  • Citizen Identity: Issues citizen identity card.
  • Employer: Verifies citizen id and issues employee card.
  • Insurance Provider: Verifies citizen id or both citizen id and employee cards depends on use case and issues insurance card.
  • Healthcare Provider: Verifies both citizen id and insurance cards.

For demo purposes, every credential will be issued in the name of Tyler Durden.
